# Secrets Manager

Get started with Bitwarden Secrets Manager and begin securely storing, sharing, and injecting infrastructure secrets into your development pipelines. Learn how to switch between Bitwarden products, add secrets, and grant machine access.

### What is Secrets Manager? (1 min)

Bitwarden Secrets Manager is a secure solution for developers and DevOps teams to store, manage, and deploy application secrets like API keys, database credentials, and certificates across their infrastructure and CI/CD pipelines.

In addition to the videos below, you can also check out the [product page](https://bitwarden.com/products/secrets-manager/) and [written documentation](https://bitwarden.com/help/secrets-manager-overview/).
