# Bitwarden Power Users

Use this article and following video guides to become a Bitwarden power user! See how you can take your Bitwarden vault beyond standard password management and use additional security functionality like Bitwarden Send, vault timeout, and biometric unlock. Learn how to organize your vault for streamlined navigation of items, set up custom fields for autofill, and more!
