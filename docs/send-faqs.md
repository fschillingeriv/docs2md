# Send FAQs

### Q: Why can't I create a file Send?

**A:** Use of text Sends is available to all Bitwarden users, however creation of file Sends is only available for premium users, including members of a paid organization (families, teams, or enterprise).

Additionally, creation of file Sends requires your email address to be verified.

### Q: Why are sends missing from my send view?

**A:** By design, Sends are ephemeral. Each created Send has a **maximum lifespan of 31 days**, configurable when you [create a Send](https://bitwarden.com/help/create-send/) or at any time by editing it. When a Send's [deletion date](https://bitwarden.com/help/send-lifespan/#deletion-date/) is reached, it will be purged from Bitwarden systems and inaccessible to both the sender and any recipients.

### Q: Can I disable Send for my organization?

**A:** Enterprise organizations can disable Send at any time using the [Remove Send policy](https://bitwarden.com/help/policies/#remove-send/). Admins and owners can implement this policy from the organization's **Settings** → **Policies** page. Enabling the policy will prevent organization members from creating or editing any Sends.
